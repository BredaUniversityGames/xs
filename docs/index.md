## About xs

**xs** is an educational game engine, made with the goal of being the smallest game making tool possible. It's designed to teach introduction to programming for game developers, who might not necessarily be programmers.

## Quick Start

1. **Download** the [latest release](https://xs-engine.itch.io/xs) from itch.io
2. **Run** the file `xs.exe` (or equivalent for your platform)
3. **Follow** the [Getting Started](getting-started.html) guide to create your first game

## What's Inside

- **Core Engine**: Written in C++ for performance
- **Wren VM**: Embedded scripting with great error messages
- **API Modules**: Math, rendering, audio, input, file I/O, and more
- **Sample Games**: Learn from included examples
- **Open Source**: Lovingly made at Breda University of Applied Sciences
