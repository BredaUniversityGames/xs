About XS

XS is an extra small, educational 2D game engine with Wren scripting. It's made with the goal of being the smallest game making tool possible. It's designed to teach introduction game engine programming at Breda University of Applied Sciences.

Wren Scripting

Uses the lightweight Wren scripting language - easy to learn, fast to run

2D Graphics

Built for pixel-perfect sprite graphics with efficient rendering

Entity-Component

Simple and powerful EC system for organizing your game logic

Audio Support

Play sound effects and background music using Fmod

Minimal UI

Clean, distraction-free interface - focus on your game

Multi-Platform

Build for PC, macOS, iOS, Nintendo Switch, and PlayStation 5

VS Code Extension

Integrated development with xs extension - visual editors, launch commands, and debug templates

Early Alpha: xs is in very early alpha and open for contribution from anyone at games@buas. Features might be missing or not be stable.

Quick Start

  1. Download the latest release from itch.io
  2. Run the file xs.exe (or equivalent for your platform)
  3. Follow the Getting Started guide to create your first game

What's Inside

  • Core Engine: Written in C++ for performance
  • Wren VM: Embedded scripting with great error messages
  • API Modules: Math, rendering, audio, input, file I/O, and more
  • Sample Games: Learn from included examples
  • Open Source: Lovingly made at Breda University of Applied Sciences
